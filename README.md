# LocaCar
 CRUD Flask/MySQL
Simple CRUD Flask.

### Technologies Used:

- Flask
- MySQL
- Html
- Bootstrap
- JavaScript

### How To Run

Clone the repository.
install flask and MySQL-Connector to venv
```
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install flask
pip install mysql-connector-python
```
Execulte the code 
```
flask run
```
